## Project Overview

This project was created for my Backend Development midterm.  
Its main purpose is to practice building a backend API that performs full CRUD operations:

- **Create** — POST  
- **Read** — GET  
- **Update** — PUT
- **Delete** — DELETE  

The project demonstrates how to structure a backend service, connect it to a database, and expose RESTful endpoints that interact with stored data.

---

## Database

The backend uses a PostgreSQL database on Render.  
The API connects to the database to store, retrieve, update, and delete records through its CRUD routes. I also made a sql template to re create the database and additional one to clear it and reset the database.
